Kilobytes Deployment

  • Java EE 6 contiene "tutto" il necessario per lo sviluppo lato server
  • I nostri deployment quindi sono molto leggeri, nell'ordine di alcuni kilobyte, durante lo sviluppo
  • I tempi di redeploy quindi possono essere nell'ordine dei millisecondi

Dynamic Code Evolution VM

  • È una JVM modificata in grado di permettere la ridefinizione delle classi caricate dal classloader, tra cui:
    • ridefinizione del corpo dei metodi (come Hotspot)
    • aggiunta e rimozione dei field e dei metodi
    • modifica alla gerarchia delle classi

  • Ovviamente si abilita solo sui server di sviluppo

Dynamic Code Evolution VM

CMD-F9 (CMD-S con IDE legacy come Eclipse) e tutte le nostre modifiche al codice sono disponibili senza dover fare redeploy durante lo sviluppo, se non in caso di cambio di configurazione applicativa
